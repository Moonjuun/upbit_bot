# upbit_bot
## 비트코인 매매 자동화 프로그램
- https://dgkim5360.tistory.com/entry/python-virtualenv-on-linux-ubuntu-and-windows
- pip3 install virtualenv
- virtualenv upbit --python=python3.8
- source upbit/bin/activate
- pip install -r requirements.txt
- python main.py
